# Site images

Place these images in `public/images/`:

- **logo.png** — Site logo (navbar and footer).  
  Use your Pirate Maxx logo (e.g. “PIRATE MAXX” / “THE SEA’S FURY”).  
  Display is ~160×48px; source can be larger (e.g. 320×96+) for sharpness.

- **hero.png** — Homepage hero background.  
  Use your full hero graphic (pirate, “GAMING • STREAMING • DESIGN • BUILDING DIGITAL BRANDS”, flames, etc.).  
  Headline and CTA buttons are overlaid on top; image shows through a dark overlay.  
  Recommended: wide image (e.g. 1920×1080 or similar) for best cover on all screens.

- **tiktok-avatar.png** (optional) — Profile image for the “From TikTok” section. In `data/tiktok.ts` set `profileImageUrl: '/images/tiktok-avatar.png'`. If omitted, a placeholder is shown.
